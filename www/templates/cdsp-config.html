CamillaDSP

CamillaDSP是一个通用的信号处理工具，用于路由和过滤声音。例如，它可以用于为有源扬声器构建交叉，或用于执行房间校正。 此版本是: $_select[version]

设置

要使用CamillaDSP，请关闭音频和多房间配置中的DSP、EQ和多房间选项。

要创建新配置，您有三个选择：
通过复制现有配置并启动camillagui（位于本页底部）即可轻松启动。
按下“新建”按钮进行最小配置，然后在其上编辑。 或者在文本编辑器中从头开始制作您自己的配置文件。对于当前选择的声音设备，这是最低限度的配置，您需要通过CamillaDSP获得任何声音：
devices: samplerate: 44100 # 如果禁用了重新采样，则不相关-那么它将被传入的采样率覆盖 chunksize: 4096 # 如果在使用更高的采样率时发生缓冲区不足，则增加 (e.g. to 8192, 16384, 32768, 65536, ...) queuelimit: 1 # 当停止播放或更改歌曲时，增加这个值会带来显著的延迟 # e.g.: samplerate=44100, chunksize=1024, queuelimit=44 => latency ~ 1s capture: type: Stdin channels: 2 # irrelevant - 将总是被覆盖 (currently always 2) format: S32LE # irrelevant - 将始终被传入的样本格式覆盖 playback: type: Alsa channels: 2 device: "$sound_device_type:$current_sound_device_number" # 默认情况下，这将在选择配置时自动更新。(正确值仅在播放停止时显示。) # format: $sound_device_sample_format # supported formats are: $sound_device_supported_sample_formats (只有在回放停止时，才会显示正确的值)
