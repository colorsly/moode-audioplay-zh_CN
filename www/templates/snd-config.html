DSP选项
MPD使用其音频缓冲区进行交叉渐变，因此必须有足够的缓冲区空间来执行请求 给定音频格式的交叉渐变秒数。
注意:设置为0将禁用交叉渐变。
700 Hz, 3.0 dB - Linkwitz
- 近似于方位角为40度的原始Linkwitz交叉馈线。最高的交叉馈电水平，最低的分离。
700 Hz, 4.5 dB - Bauer
- 在大约3米的距离上近似方位角为30度的虚拟扬声器位置。
800 Hz, 6.0 dB - Chu Moy
- 近似于Chu Moy改进的Linkwitz交叉馈线。
650 Hz, 10.0 dB - Jan Meier
- 近似于Jan Meier自然交叉馈送。最低的交叉馈电电平，最高的分离度。

均衡器

均衡器由MPD、AirPlay和Spotify连接渲染器支持。如果MPD音频输出设置为“Bluetooh speaker”（蓝牙扬声器）或“Multiroom Sender”（多房间发送器）或“Receiver”（接收器）打开，则它们不可用。 请注意：均衡器、交叉馈送和极性反转是互斥的。启用其中一个将禁用另一个上的设置按钮。

CamillaDSP是一个通用的信号处理工具，用于路由和过滤声音。例如，它可以用于为有源扬声器构建交叉，或用于执行房间校正。CamillaDSP由Henrik Enquist维护。与moOde的集成是由@bitlab完成的。
Feature summary:
- IIR filters (BiQuad)
- FIR filters (Convolution via FFT)
- Built-in sample rate converter
- Filters can be chained freely
- Flexible routing
- Simple YAML configuration
- All calculations done with 64-bit floats
- ARM Neon 64-bit instruction set is used
- Pipeline editor with expert mode
